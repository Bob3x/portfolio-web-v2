Über mich

Ich heiße Borislav Ginov – mein Weg führte mich von der Kunst zur Technik, von Bulgarien nach Berlin, von der Innenarchitektur zur Webentwicklung. Meine kreative Reise begann mit einem Bachelor in Innenarchitektur und einem Master in Digitalen Künsten. In Berlin tauchte ich in die Welt der 3D-Visualisierung ein – Räume gestalten, Produkte zum Leben erwecken, Visionen greifbar machen. Dabei habe ich viel über Designprozesse, Teamarbeit und kreative Projektentwicklung gelernt.
Doch ich wollte mehr: Systeme verstehen, digitale Erlebnisse formen, Ideen zum Funktionieren bringen. So landete ich in der Webentwicklung – dem Ort, an dem Design und Technik perfekt zusammenspielen. Heute entwickle ich digitale Lösungen, die nicht nur funktionieren, sondern auch inspirieren – mit einem klaren Blick für Ästhetik und einem starken Sinn fürs Detail.

Projekte

PokedexXx

JavaScript CSS HTML GitHub

Pokédex ist eine einfache Webanwendung, entwickelt mit HTML, CSS und JavaScript. Nutzer*innen können durch eine Liste von Pokémon-Karten blättern und gezielt nach Namen suchen. Beim Klick auf ein Pokémon-Bild werden weitere Details zum Charakter in einem individuellen Modal angezeigt. Die Anwendung bietet ein übersichtliches, benutzerfreundliches UI und ruft Daten über die offizielle Pokémon-API ab – gefiltert auf ausgewählte Elemente.

bestflix API

Node Express MongoDB Swagger

RESTful API zur Verwaltung einer Filmdatenbank mit Nutzer-Authentifizierung und -Autorisierung. Die API-Endpunkte und Routen wurden mit dem Express-Framework und Middleware-Funktionen definiert. Die NoSQL-Datenbank wurde mit Mongoose erstellt und um sichere Prozesse ergänzt – darunter Autorisierung, JWT-Authentifizierung und Datenvalidierung.

bestflix frontend - Angular

meeet app

React HTML CSS Gherkin Google API Data Visualisation

meeet ist eine React-Anwendung, entwickelt nach TDD- und BDD-Prinzipien, und mit dem Serverless Framework auf AWS Lambda bereitgestellt. Es handelt sich um eine moderne, skalierbare Progressive Web App (PWA) mit spannenden Features. Nutzer*innen können Veranstaltungen in der gewählten Stadt durchsuchen. Die Eventdaten werden über die Google Calendar API abgerufen.
